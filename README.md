# qr

> Simple, ad-free QR code generator

I built this because it seems when you Google "QR code generator", all the top results either have a bunch of ads, or require an account for some (probably shady) reason.

After my wife asking me enough times what site to use for QR codes, I threw this together, deployed to GitHub Pages ✨
